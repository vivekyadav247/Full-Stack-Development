### **Fonts**

- **Font Family:** `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`
- **Font Weights Used:**
  - 400: Regular body text
  - 500: Medium emphasis (labels, secondary headings)
  - 600: Bold emphasis (main headings, titles)

### **Color Palette**

| Element            | Color                              | Usage                                     |
| ------------------ | ---------------------------------- | ----------------------------------------- |
| **Background**     | `#ffffff` (White)                  | Main page background                      |
| **Text Primary**   | `#000000` (Black)                  | Body text, headings                       |
| **Text Secondary** | `#666666` (Dark Gray)              | Secondary text, captions                  |
| **Accent Color**   | `rgb(24, 226, 153)` (Teal/Cyan)    | Buttons, highlights, badges, hover states |
| **Dark Navy**      | `lab(2.42579 -0.155291 -0.470081)` | Dark backgrounds, subtle accents          |
| **Light Overlay**  | `rgba(100, 100, 100, 0.15)`        | Semi-transparent backgrounds for cards    |
| **Border Color**   | `rgba(0, 0, 0, 0.1)`               | Subtle borders and separators             |

### **Key Color Usage**

- **Teal Accent (`rgb(24, 226, 153)`):** Used for:
  - "New" badge
  - Feature labels ("LLMS.TXT & MCP")
  - "Enterprise-reinvention" label
  - Hover states on social links
  - Primary action buttons in dark sections

- **Navy Background (`lab(2.42579 -0.155291 -0.470081)`):** Used for:
  - Dark overlay backgrounds
  - Enterprise section background
  - Footer background
